# Amplificación en alta frecuencia (GHz)
Investigación realizada durante una pre-professional practice para el Millennium Institute for Research in Optics.
Este repositorio contiene documentos de la investigación (simulaciones e informes).

Disclaimer:
Algunas simulaciones ocupan componentes que no son base de LTspice, sino que se importan los _scattering parameters_ de distintos lugares, y se traducen a un objeto simulable en LTspice mediante el programa [s2spice](https://github.com/transmitterdan/s2spice). Intentar realizar estas simulaciones sin tener los componentes adecuados resultará en un error.
